## 用例1：ZL105A T6 固溶温度查询

**输入**：`ZL105A T6固溶温度是多少`

**期望**：返回固溶温度 520~530°C、保温时间 6~18h、时效 170~180°C/8~12h

**通过条件**：输出包含 `520~530°C` 且包含 `6~18h`

---

## 用例2：X射线3个气孔（8mm厚度）

**输入**：`X射线发现3个气孔，铸件厚度8毫米`

**期望**：❌ 拒收（8mm厚度 I类铸件允许≤1个气孔）

**通过条件**：输出包含 `拒收` 且包含 `超标`

---

## 用例3：X射线1个缩孔（8mm厚度）

**输入**：`X射线发现了1个缩孔，铸件厚度8毫米`

**期望**：❌ 拒收（缩孔零容忍，发现即拒收）

**通过条件**：输出包含 `拒收` 且包含 `零容忍`

---

## 用例4：合金未知时温度超标（545°C）

**输入**：`固溶炉温度到545度了`

**期望**：提示补充合金牌号（不能在没有合金信息的情况下用默认545°C判断）

**通过条件**：输出包含 `请补充` 或 `需要` + `合金`

## 用例5：ZL105A 固溶炉 545°C 超标

**输入**：`ZL105A固溶炉温度到545度了`

**期望**：⚠️ 严重/必须立即停止（ZL105A固溶上限525°C，超标25°C有过烧风险）

**通过条件**：输出包含 `严重` 或 `必须立即停止`

## 回归测试命令（全部 10 个）

```bash
cd /root/.hermes/skills/data-science/aircraft-casting-expert && python3 -c "
from diagnose import diagnose
from render_card import render

cases = [
    ('ZL105A T6固溶温度是多少', ['520~530', '6~18h'], '工艺参数'),
    ('X射线发现3个气孔，铸件厚度8毫米', ['拒收', '超标'], '缺陷-气孔拒收'),
    ('X射线发现了1个缩孔，铸件厚度8毫米', ['拒收', '零容忍'], '缺陷-缩孔零容忍'),
    ('固溶炉温度到545度了', ['请补充', '合金'], '合金缺失保护'),
    ('ZL105A固溶炉温度到545度了', ['严重', '立即停止'], '温度超标严重'),
    ('ZL205A T6要做成I类铸件，力学性能要达到多少', ['420', '450'], '力学性能查询'),
    ('ZL205A T7要怎么取样检验', ['晶间腐蚀', '荧光'], '取样检验规则'),
    ('ZL114A固溶参数', ['≤8s'], 'ZL114A参数-无重复前缀'),
    ('什么是T6状态', ['💡', '📌', 'T6'], '术语查询-T6状态'),
    ('固溶处理是什么意思', ['💡', '加热', '淬火'], '术语查询-固溶处理'),
]

passed = 0
for question, keywords, label in cases:
    result = diagnose(question)
    text = render(result)
    ok = all(k in text for k in keywords)
    status = '✅' if ok else '❌'
    if not ok:
        print(f'{status} {label}')
        print(f'   问题: {question}')
        print(f'   期望含: {keywords}')
        print(f'   实际: {text[:200]}')
    passed += 1 if ok else 0

print(f'结果: {passed}/{len(cases)} 通过')
"
```

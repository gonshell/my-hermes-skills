# 段落级阅读体验问题清单

本文档是 `chapter-consistency-checker` 的补充检查项，专门针对段落级的AI味阅读体验问题。

## 常见问题模式

### P1. 孤立短段（独立成行的短句段落）

**识别：** 某段落只有1-2句话，字数<20字，与上下文形成突兀断连。

**典型案例：**
```
这段的问题是：[一段完整论述]

这些失误有一个共同的名字：盲区。   ← 孤立短段

这不是偶然的缺陷。   ← 同上
```

**修复：** 合并到前一段末尾，或合并到后一段开头。

---

### P2. 设问+定义+排比+总结 机械三连

**识别：** 小节内部出现"设问句→加粗定义→排比例子→短句总结"的公式化结构。

**典型案例：**
```
然而，符号主义有一个从根源上无法解决的难题。   ← 设问（表面是问，实际自答）

**常识知识如何获取？**   ← 加粗定义

"水是湿的"。"父亲比儿子大"。"如果你关上一扇打开的门……"   ← 排比

这是符号主义绕不过去的难题。   ← 短句总结
```

**修复：** 取消设问，直接陈述论点；取消总结句，让例子自己说话。

---

### P3. "更X的是"过渡句

**识别：** 段内突然出现"更重要的是"、"更有趣的是"、"更根本的是"等过渡词。

**修复：** 直接陈述第二论点，不加过渡词标记。

---

### P4. "同等X的Y"刻意平行句

**识别：** "另一个同等根本的困难是"、"另一个同等严重的困境是"等刻意与上段平行的结构。

**修复：** 改为"还有……的困难"或"……是另一个问题"，消除人为平行感。

---

### P5. "这是X的Y"机械断言

**识别：** "这是搜索决策问题"、"这是不舒服的妥协"等以"这是"开头的机械判定句。

**修复：** 改为"这里涉及的是……"、"属于……范畴"。

---

### P6. 章节末尾短句总结

**识别：** 章节最后一段突然出现一行单独成段、字数<15的总结性短句。

**典型案例：**
```
[完整论述]

但方向是正确的。   ← 突兀短句
```

**修复：** 合并到前一段末尾作为自然结语，或直接删除。

---

## 扫描方法

1. 将文章按段落拆分（双换行分隔）
2. 标记所有字数<20的段落 → 逐一检查是否孤立短段
3. 搜索"**XX？**"模式 → 检查后续是否有"XX是……"总结句
4. 搜索"同等X的Y"、"更X的是" → 定位刻意平行/过渡结构
5. 搜索以"这是"开头的独立句子 → 定位机械断言
6. 在章节末尾前后各三段，检查是否有字数<15的孤立收尾句

## 修复优先级

| 优先级 | 问题类型 | 修复必要性 |
|--------|---------|-----------|
| P1 | 孤立短段 | 必须修复，影响阅读节奏 |
| P2 | 设问三连 | 必须修复，AI味最重 |
| P3 | 过渡句 | 建议修复 |
| P4 | 平行句 | 建议修复 |
| P5 | 机械断言 | 建议修复 |
| P6 | 末尾短句 | 必须修复，影响阅读节奏 |

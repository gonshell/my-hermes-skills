# 多章节一致性检查器

## 概述

对多章节长篇文档进行**单Agent深度一致性扫描**，无需多Agent讨论。通过逐维度比对，发现跨章节的术语矛盾、语气漂移、细节层次不均、人称混乱、引用格式不一致等问题。

## 触发条件

- 用户提供多个章节内容（Markdown/纯文本），要求检查一致性
- 用户正在写作多章节作品（小说、技术书、论文、系列博客），希望发现不统一之处
- 关键词：「一致性检查」「章节矛盾」「术语统一」「前后不一致」

## 检查维度（5大维度）

### 1. 术语一致性（Terminology）

- 同一概念是否使用不同译名/名称（如「机器学习」vs「machine learning」vs「ML」）
- 专业术语首次出现是否有定义，后续是否保持统一
- 缩写使用是否一致（首次全称+缩写，之后统一用缩写）
- 技术名词拼写是否一致（英/中/混用）

### 2. 语气一致性（Tone）

- 各章节语气是否与整体定位一致（学术/通俗/对话/叙事）
- 是否出现突然的语气切换（如正式章节中突然口语化）
- 称呼读者方式是否统一（「你」vs「您」vs「读者」vs 隐含）
- 感叹号、问号等情感标点使用风格是否统一

### 3. 细节层次一致性（Detail Level）

- 相似主题的章节是否在深度上保持一致
- 是否有章节过度展开而另一些过于简略
- 示例/代码/图表的丰富程度是否均衡
- 章节篇幅是否与内容重要性匹配

### 4. 人称一致性（Person）

- 叙述人称是否统一（第一人称「我」/ 第三人称「作者」/ 第二人称「你」）
- 是否在同一章节内人称切换（如从「我们」跳到「我」）
- 被动语态 vs 主动语态的使用比例是否一致
- 引用自身经验时的人称是否前后统一

### 5. 引用一致性（Citation）

- 引用格式是否统一（脚注/尾注/行内引用/链接）
- 参考文献列表格式是否一致（APA/IEEE/GB/T 7714 等）
- 同一引用在不同章节是否使用相同格式
- 链接引用 vs 文字引用的混合使用是否合理
- 图片/表格编号是否连续且格式统一

## 执行步骤

### 步骤 1：收集全部章节

```
将所有章节内容按顺序排列，标记章节编号和标题。
确认章节总数量和预期总篇幅。
```

### 步骤 2：逐维度扫描

对每个维度分别执行：

1. **提取特征**：从每章提取该维度相关特征（如所有术语、语气关键词、人称代词）
2. **跨章比对**：章节间两两比对，记录差异
3. **标注位置**：记录问题出现的章节和段落位置
4. **严重性分级**：
   - 🔴 **严重**：概念矛盾、术语完全不同、人称混乱
   - 🟡 **中等**：语气漂移、细节层次不均、格式不完全一致
   - 🟢 **轻微**：偶发不一致、风格细微差异

### 步骤 3：生成检查报告

输出结构化报告，格式如下：

```markdown
# 一致性检查报告

## 概览
- 检查章节数：X
- 发现问题总数：X（🔴 X | 🟡 X | 🟢 X）
- 最需关注的维度：[维度名]

---

## 1. 术语一致性
| 问题 | 章节 | 严重性 | 建议 |
|------|------|--------|------|
| 「数据库」vs「DB」混用 | Ch2¶3, Ch5¶1 | 🟡 | 统一为「数据库」 |
| ... | ... | ... | ... |

## 2. 语气一致性
[同类表格]

## 3. 细节层次一致性
[同类表格]

## 4. 人称一致性
[同类表格]

## 5. 引用一致性
[同类表格]

---

## 修改建议优先级
1. [最高优先级的修改建议]
2. [次高优先级的修改建议]
3. ...
```

### 步骤 4：提供修正方案

对每个问题给出：
- 问题的具体位置（章节号 + 段落）
- 推荐的统一方案
- 需要全局替换的术语表（如适用）

## 注意事项

- **不做主观评价**：只报告不一致，不评判哪种写法更好
- **区分有意vs无意**：作者可能有意在不同章节使用不同语气（如教程vs参考），需标注
- **保留原文风格**：修正建议应尊重原文整体风格
- **检查范围可配置**：用户可选择只检查部分维度

## 输出格式

默认输出 Markdown 格式报告。如用户需要，可额外输出：
- JSON 格式（用于程序化处理）
- 差异对照表（原文 vs 建议修改）
- 术语统一词典

## 示例用法

用户输入：
```
请检查以下3个章节的一致性：
[Chapter 1 内容]
[Chapter 2 内容]
[Chapter 3 内容]
```

或分步提供：
```
第一章一致性基线建立 → 后续章节逐一比对
```

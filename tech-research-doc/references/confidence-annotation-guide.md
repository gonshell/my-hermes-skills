# 置信度标注指南

## 两套标注体系的关系

| 体系 | 标注什么 | 何时使用 |
|------|---------|---------|
| **[N/T/E]** | 数据**来源**的可信度 | 所有类型，标注具体数据点 |
| **置信度分级** | **判断**的可信程度 | 分析/预测类内容 |

两套体系独立使用，互不替代。一个 `[N]` 官方数据支撑的结论，其推断部分仍可能是 `[HCF]`。

---

## [N/T/E] 数据来源标注

| 标注 | 含义 | 示例 |
|------|------|------|
| `[N]` | 官方发布 | NVIDIA Datasheet、RFC 文档、官方公告 |
| `[T]` | 行业经验 | 多方印证的估算、从业者共识 |
| `[E]` | 估算 | 基于已知信息的推算、填补空缺 |

### 标注要求

- `[N]` 必须附来源 URL 或文档名
- `[T]` 必须注明"非官方"并列出信源数量
- `[E]` 必须写明推断依据和计算方式

### 常见错误

```
❌ "昇腾 910B 算力 ~320 TFLOPS [N]"
   → 昇腾无官方 Datasheet，这不是 [N]

✅ "昇腾 910B 算力 ~320 TFLOPS [T]（行业广泛引用值，3个独立信源交叉验证）"

❌ "预计2027年市场规模达到500亿 [N]"
   → 预测值不可能是 [N]

✅ "IDC 预测2027年市场规模达到500亿 [T]（引用 IDC 2025Q2 报告，属行业预估）"
```

## 置信度分级

### L1 事实 `[FACT]`

**定义**：公开可验证的数据或事件。

**允许的报告类型**：所有。

**要求**：必须附来源 URL 或引用。

**示例**：
```
H100 SXM5 HBM3 带宽为 3.35 TB/s [FACT][N]
（来源：NVIDIA H100 Datasheet, p.7, https://resources.nvidia.com/...）
```

### L2 高置信推断 `[HCF]`

**定义**：≥2 个独立信源得出一致结论。

**允许的报告类型**：所有。

**要求**：必须列出信源。

**示例**：
```
昇腾 910B 推理吞吐约为 H100 的 1/3 [HCF]
（信源1：MLPerf Inference v4.0 官方结果 [N]
 信源2：华为开发者社区 benchmark [T]
 信源3：第三方测评实验室报告 [T]）
```

### L3 合理推测 `[REASON]`

**定义**：基于已知事实的逻辑推理，缺少直接证据。

**允许的报告类型**：深度分析、趋势研判。

**要求**：必须写明推理链。

**示例**：
```
昇腾 910C 若采用 HBM3e，带宽瓶颈可缓解 2-3x [REASON]
推理链：
  前提1：910B 的主要瓶颈在 HBM2 带宽 ~900 GB/s [FACT]
  前提2：HBM3e 带宽密度约为 HBM2 的 2.5-3x [FACT]（SK Hynix 公开数据）
  推理：若 910C 采用 HBM3e，带宽可提升至 ~2.2-2.7 TB/s
  注意：910C 是否采用 HBM3e 尚无官方确认 [E]
```

### L4 趋势外推 `[TREND]`

**定义**：基于历史数据的线性或非线性外推。

**允许的报告类型**：仅趋势研判。

**要求**：必须标注时间范围和核心假设。

**示例**：
```
预计 2027 年国产 AI 芯片市场份额可达 15-20% [TREND]
时间范围：2024-2027
核心假设：
  1. 美国出口管制政策不变
  2. 国产芯片年性能提升率维持在 40-60%
  3. 国内大模型训练需求年增长率 ≥ 50%
历史数据基线：
  2024: ~5%（IDC 数据）[N]
  2025: ~8%（行业预估）[T]
```

### L5 个人判断 `[JUDGMENT]`

**定义**：经验性判断，无定量支撑。

**允许的报告类型**：仅在明确标注时使用。

**要求**：必须声明"非定量支撑"。

**关键规则**：**不允许作为核心结论的支撑**。只能用于辅助说明、背景补充。

**示例**：
```
从工程实践角度看，昇腾生态的工具链成熟度还需 1-2 年追赶 [JUDGMENT]
（基于社区反馈和开发者体验综合判断，非定量分析）
```

## 混合使用规则

### 同一段落中的级别不一致

一段话中涉及多个级别时，**以最低级别为准**标注该段落的整体结论。

```
H100 带宽 3.35 TB/s [FACT]，
昇腾带宽约 900 GB/s [HCF]，
因此差距约 3.7x。
→ 这段结论的置信度为 [HCF]（取较低者）
```

### [N/T/E] 与置信度的交互

| 组合 | 含义 | 合理性 |
|------|------|--------|
| [FACT][N] | 官方确认的事实 | ✅ 最强 |
| [FACT][T] | 行业公认的事实 | ✅ 可接受 |
| [FACT][E] | 矛盾：估算不可能是事实 | ❌ 不允许 |
| [HCF][N] | 官方数据支撑的推断 | ✅ |
| [HCF][T] | 行业数据支撑的推断 | ✅ |
| [REASON][N] | 基于官方数据的推理 | ✅ |
| [REASON][T] | 基于行业数据的推理 | ✅ |
| [TREND][E] | 基于估算的外推 | ⚠️ 需特别声明假设 |

### 按报告类型的最大允许级别

| 报告类型 | 最大允许级别 | 原因 |
|---------|------------|------|
| 技术调研 | `[HCF]` | 调研重事实，不重预测 |
| 竞品对比 | `[HCF]` | 对比需可验证，不能靠推测 |
| 状态评估 | `[HCF]` | 评估需数据支撑 |
| 深度分析 | `[REASON]` | 分析允许逻辑推理 |
| 趋势研判 | `[TREND]` | 预测需要外推，但必须声明假设 |
| 方案设计 | `[HCF]` | 方案需基于事实，工期估算用 [E/T] |

# 报告结构模板

## 文件命名

`{股票名}_{时间窗口}_深度分析.md`

示例：
- `宇通客车_半年_深度分析.md`
- `比亚迪_一年_深度分析.md`
- `宁德时代_半年_深度分析.md`

## 报告结构（8 章 + 3 附录）

```

### 1.2 投资判断概要
### 1.3 三大核心风险
```

## 各章节核心要点

### 第 1 章：执行摘要
- **3 句话核心结论**（每个不超过 50 字）
- **投资判断概要**（表格：维度/判断/置信度）
- **三大核心风险**（按风险大小排序）

### 第 2 章：公司概况
- **基本情况**（股票代码、上市时间、总股本、实际控制人、行业地位）
- **业务结构**（最新年份，按收入来源拆分）

### 第 3 章：股价走势
- **价格骨架表**（关键节点 5-10 个）
- **关键行情数据**（振幅、跑输/跑赢指数）
- **估值迁移**（PE 变化路径）
- **关键月度数据**（如果是制造/消费）

### 第 4 章：涨跌动力
- **上涨动力**（三层：故事/兑现/催化）
- **下跌动力**（三层反向）
- **反常断言**（最反常识的发现）

### 第 5 章：护城河与竞争
- **市场层面**（波特五力）
- **公司层面**（4 类护城河：成本/品牌/转换/网络）
- **横向对比表**（vs 同行 + 全球巨头）
- **可证伪测试**（每个护城河都有反证条件）

### 第 6 章：风险与替代威胁
- **风险矩阵**（风险/触发条件/概率/影响）
- **替代威胁 4 个层次**（直接竞品/跨界/技术/需求）
- **关键监控指标**（定期检查的指标）

### 第 7 章：估值参考
- **方法适用性评估**（每种方法适用/有限适用/不适用）
- **估值参考区间**（不给目标价）
- **情景分析**（6 个情景）
- **概率加权**（期望收益）
- **重要免责声明**

### 第 8 章：方法论
- **数据来源**（按类别列出）
- **方法论说明**（用了什么方法）
- **重要局限性**（数据/方法/判断/使用）
- **核心反思**（研究过程中的反思）

## 写作风格规范

### 应该用

- 具体数字（"营收 414.26 亿"）
- 对比基准（"vs 金龙 8.23 亿"）
- 区间（"估值 25-40 元"）
- 反证测试（"什么情况下不成立"）
- 可靠度标注（"披露数据""估算数据"）

### 不应该用

- 主观形容词（"很深的""强大的""领先的"）
- 精确目标价（"目标价 40 元"）
- 营销话术（"行业首个""国内独一档"）
- 权威断言（"专家说""分析师认为"）
- 必然性表述（"未来会涨""必然实现"）

## 文件长度建议

- **每个章节**：500-1500 字
- **整个报告**：15000-25000 字
- **执行摘要**：500-1000 字

## 报告保存路径

- **本地**：保存到工作区根目录
- **飞书**（可选）：同步到飞书文档，URL 返回给用户

## 报告完成后的交付清单

```
[ ] 本地 Markdown 文件
[ ] 飞书文档 URL（如果用户要求）
[ ] 关键数字清单（股价、估值、风险点）
[ ] 重要免责声明
```

## 后续建议

1. **定期更新**：每季度根据新财报更新报告
2. **持续跟踪**：根据报告的"关键监控指标"持续跟踪公司
3. **对比同行**：可以批量应用此 SKILL 分析同行业其他公司
4. **形成组合**：用此 SKILL 分析的多家公司组成投资组合候选

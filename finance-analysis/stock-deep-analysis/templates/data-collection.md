## 数据标注规范

每个数据点必须包含：

```
[数据]：[数值]
[来源]：[具体来源]
[时点]：[具体日期]
[可靠度]：[高/中/中低/低]
[类型]：[披露/估算/营销话术]
```

## 数据来源优先级

1. **公司官方**（年报、季报、公告、投资者关系）
2. **交易所**（上交所、深交所、港交所）
3. **行业协会**（中客网、中汽协、中商网）
4. **第三方数据**（同花顺、东方财富、Wind、iFinD）
5. **卖方研报**（东吴、长江、中信、国信）

每个数据至少 2 个来源验证。

## 数据时效性检查

每个数据必须标注：
- **时点**：数据对应的具体日期
- **截止**：本次分析的时间窗口
- **过期风险**：如果数据超过 6 个月，需要更新

## 数据采集常见错误

| 错误 | 防范 |
|---|---|
| 单一来源 | 至少 2 个来源验证 |
| 不标注时点 | 每个数据都标注具体日期 |
| 估算当披露 | 明确标注估算 vs 披露 |
| 引用过期数据 | 标注数据截止日期 |
| 用营销话术 | 标注"公司宣称" |

# 5 阶段详细方法

## 阶段 1：基础数据采集的详细操作

### 1.1 数据采集清单（强制）

按以下顺序采集，缺一不可：

```
[必查] 财务数据
- 最新年报：营收 / 归母净利 / EPS / 经营现金流 / ROE
- 最新季报：单季同比 / 环比变化
- 历史 3 年财务数据（用于趋势分析）

[必查] 销量数据（如适用）
- 公司产销快报：月度数据
- 同业销量对比

[必查] 行业数据
- 行业总规模（用行业协会数据）
- 公司市占率（用第三方数据）
- 行业趋势（最近 3 年增长情况）

[必查] 同行对比
- 至少 2 家同行（同业务线）
- 关键指标：营收 / 净利 / 毛利率 / 海外占比 / 研发占比

[如有] 全球巨头对比
- 全球前三名（如 Daimler / Toyota / Bosch 等）
- 用于国际竞争力对比

[必查] 政策环境
- 行业相关政策（最近 2 年）
- 监管动作（如行业准入）
- 补贴政策
```

### 1.2 数据采集的搜索方法

按以下顺序搜索，每个数据至少 2 个来源验证：

1. **公司官方**：年报 / 季报 / 公告 / 投资者关系记录表
2. **交易所**：上交所 / 深交所 / 港交所 披露
3. **行业协会**：中客网 / 中汽协 / 中商网
4. **第三方数据**：同花顺 / 东方财富 / Wind / iFinD
5. **卖方研报**：东吴 / 长江 / 中信 / 国信（用于交叉验证）

### 1.3 数据标注规范

每个数据点必须包含：

```
[数据]：[数值]
[来源]：[具体来源]
[时点]：[具体日期]
[可靠度]：[高/中/中低/低]
[类型]：[披露/估算/营销话术]
```

**示例**：
```
[数据]：2025 营收 414.26 亿元
[来源]：宇通客车 2025 年年度报告
[时点]：2026-03-30 披露
[可靠度]：高
[类型]：披露
```

## 阶段 2：时点走势分析的详细操作

### 2.1 关键节点的识别方法

按以下逻辑识别关键节点（半年内通常 5-10 个）：

**必识别节点**：
1. 期初 / 期末
2. 期间最高 / 最低
3. 财报披露日（业绩拐点）
4. 重大事件（中标 / 收购 / 政策变化）
5. 估值转折点（PE 明显变化）

**节点标注的 5 个字段**：
1. 日期（具体年月日）
2. 价格（具体到分）
3. 事件（触发原因）
4. 性质（高点/低点/转折点）
5. 数据来源

### 2.2 估值迁移的计算方法

**PE-TTM 公式**：当前股价 / 过去 12 个月摊薄每股收益

**估值迁移表格**：

| 时点 | 股价 | EPS-TTM | PE-TTM | 推动力 |
|---|---|---|---|---|
| 2025-03 | 26.83 元 | 1.49 元 | 18.0 倍 | 2024 净利 +126.5% |
| 2026-04-01 | 37.24 元 | 1.49 元 | 25.0 倍 | 海外首超国内催化 |
| 2026-06-18 | 26.04 元 | 1.49 元 | 12-13 倍 | 估值消化回归 |

### 2.3 指数对比方法

**对比基准**（按股票市场选择）：
- A 股：上证指数 / 沪深 300 / 申万一级行业指数
- 港股：恒生指数 / 国企指数
- 美股：标普 500 / 道琼斯 / 纳斯达克

**对比方式**：
- 同期相对涨幅（股票涨幅 vs 指数涨幅）
- 跑赢/跑输的百分点
- 在历史区间中的相对位置

## 阶段 3：涨跌动力分析的详细操作

### 3.1 三层叠加模型的具体应用

**上涨动力三层**：
- **故事层**（1-3 年）：长期叙事，业绩验证前的预期
- **兑现层**（6-12 个月）：业绩兑现，预期变现实
- **催化层**（1-3 个月）：事件触发，加速上涨

**下跌动力三层（反向）**：
- **业绩兑现失败**：增速不达预期
- **叙事动摇**：长期逻辑被质疑
- **估值消化**：前期涨幅过大，估值回归

### 3.2 反常断言识别清单

满足以下任一条件即为反常断言：

| 反常类型 | 识别条件 |
|---|---|
| 业绩下滑但质量改善 | 净利同比下降，但毛利率/现金流/资产质量改善 |
| 估值下跌但基本面未变 | 股价大幅下跌，但 EPS 同比未变（情绪驱动） |
| 行业下跌但该公司上涨 | 同业普遍下跌，该公司逆势上涨（独立 alpha） |
| 行业上涨但该公司下跌 | 同业普遍上涨，该公司逆势下跌（独立风险） |
| 涨价但销量上升 | 产品涨价但销量同时上升（定价权强） |
| 降价但毛利上升 | 产品降价但毛利率上升（成本下降更快） |

### 3.3 动力分析表模板

| 层次 | 时点 | 核心数据 | 市场逻辑 | 强度 |
|---|---|---|---|---|
| 故事层 | 时间范围 | 数据 | 逻辑 | ★~★★★ |
| 兑现层 | 时间范围 | 数据 | 逻辑 | ★~★★★ |
| 催化层 | 时间范围 | 数据 | 逻辑 | ★~★★★ |

## 阶段 4：护城河与竞争分析的详细操作

### 4.1 双重视角的具体操作

**视角 1：行业层面（波特五力）**

每种力量的分析维度：
- 力量强度（高/中/低）
- 关键事实（数据）
- 公司应对方式
- 评估

**视角 2：公司层面（4 类护城河）**

每类护城河的分析模板：
```
护城河类型：[成本/品牌/转换/网络]

核心问题：[什么问题最难复制？]

客观证据：
- 数据点 1（来源）
- 数据点 2（来源）
- 数据点 3（来源）

强度评估：[强/中/弱]

可证伪测试：
- [什么情况下这道理会不成立？]
- 条件 1
- 条件 2

关键监控指标：
- 指标 1（如何监测弱化）
- 指标 2
```

### 4.2 横向对比的强制表头

每张对比表必须有：

| 维度 | 标的 | 同行 1 | 同行 2 | 全球巨头 |
|---|---|---|---|---|
| 数据 1 | X | Y | Z | W |
| 数据 2 | X | Y | Z | W |

不允许只列出标的公司的数据而不列出对比对象。

### 4.3 5 道护城河 vs 4 类护城河

**5 道护城河**（旧框架，按可复制性）：
1. 海外销售服务网络
2. 规模效应
3. 品牌 + 高端市场认知
4. 电动化技术先发
5. 分红 + 国资治理

**4 类护城河**（新框架，按护城河类型）：
1. 成本优势
2. 品牌资产
3. 转换成本
4. 网络效应

**推荐使用 4 类框架**（更清晰、可证伪）。

## 阶段 5：估值参考的详细操作

### 5.1 估值方法适用性评估

每个估值方法都要先评估是否适用：

```
方法：[PE/DDM/EV-EBITDA/DCF]

适用条件：
- [ ] 业绩稳定增长（Y/N）
- [ ] 可比公司多（Y/N）
- [ ] 关键数据可信（Y/N）
- [ ] 假设合理（Y/N）

评估结论：[适用/有限适用/不适用]
```

### 5.2 情景分析的具体操作

**步骤 1：确定情景数量**
- 至少 4 个情景（牛 / 偏牛 / 偏熊 / 熊）
- 建议 6 个（牛 A / 牛 B / 基 A / 基 B / 熊 A / 熊 B）

**步骤 2：每个情景的关键参数**
- 净利增长率
- 假设 PE 倍数
- 估值区间（用 PE × EPS 计算）

**步骤 3：每个情景的概率**
- 标注"主观判断"
- 给出概率区间的依据（即使是主观也要说明）

**步骤 4：期望收益**
- 期望收益 = Σ(概率 × 收益)

### 5.3 估值区间的形成逻辑

**下沿**（保守）：
- 用最不利的情景
- 用最低的 PE 倍数

**上沿**（乐观）：
- 用最有利的情景
- 用最高的 PE 倍数

**中间值**（中性）：
- 用概率加权
- 用历史均值

### 5.4 估值的关键免责声明模板

```
本估值分析存在以下局限性：

1. 数据层面：
   - 部分数据（如 EBITDA）是估算而非披露
   - 同行数据可能不完整
   - 部分数据已过期

2. 方法层面：
   - 多数估值方法不适用于该公司（如高增长 + 高分红股票）
   - 6 个样本的回测无统计显著性

3. 判断层面：
   - 情景概率是主观叙事概率，不是统计概率
   - 估值区间宽度是经验估计，不是精确计算

4. 使用层面：
   - 本报告不构成投资建议
   - 估值 ≠ 预测，市场可能长期偏离估值区间
   - 任何决策应结合实时数据
```

## 阶段 6：反思机制的详细操作

### 6.1 每个阶段的反思清单

**阶段 1 反思**：
- [ ] 数据来源是否独立可查？
- [ ] 是否区分了披露 vs 估算？
- [ ] 是否有数据时效性问题？

**阶段 2 反思**：
- [ ] 价格对比是否有基准（不是孤立判断）？
- [ ] 估值迁移是否给了具体 PE？
- [ ] 是否漏掉了关键节点？

**阶段 3 反思**：
- [ ] 上涨/下跌动力是否对等分析（不是只看一边）？
- [ ] 反常断言是否有数据支撑？
- [ ] 是否陷入"故事陷阱"？

**阶段 4 反思**：
- [ ] 横向对比是否有数据（不是主观"领先"）？
- [ ] 可证伪测试是否明确？
- [ ] 是否陷入"营销话术"陷阱？

**阶段 5 反思**：
- [ ] 估值方法适用性是否评估？
- [ ] 是否给区间而不是目标价？
- [ ] 是否标注了不确定性？

### 6.2 整个 SKILL 的最终自检

完成后回答：

1. 是否有任何"领先""超越""最强"无对比基准？
2. 是否有任何核心结论无反证条件？
3. 是否有任何数据无来源标注？
4. 是否有任何主观判断未明确标注？
5. 是否有任何估值给精确数字而非区间？

如果任何一项为"是"，立即修正。
